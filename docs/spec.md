# CCTV-17 收视日报 PPT 各页 SPEC

> 幻灯片尺寸：13.33 × 7.5 英寸（Emu 12192000 × 6858000）
> 全局字体：微软雅黑（Microsoft YaHei）
> 生成模式：简版（6-sheet Excel → 11 页） / 标准版（12-sheet → 最多 17 页）

---

## 通用约定

### 变化描述口径
| 条件 | 描述 |
|---|---|
| 变化幅度 = 0 | `保持不变` |
| \|变化幅度\| < 5% | `基本持平` |
| \|变化幅度\| ≥ 5%，上升 | `提升xx%` |
| \|变化幅度\| ≥ 5%，下降 | `下降xx%` |

### 排名变化描述
| 条件 | 描述 |
|---|---|
| 排名上升 | `上升N位` |
| 排名下降 | `下滑N位` |
| 排名不变 | `保持不变` |

### 颜色规范（代码控制部分）
| 用途 | 颜色 |
|---|---|
| 提升/上涨 | 红色 `#FF0000` |
| 下降/下跌 | 绿色 `#00B050` |
| CCTV-17 高亮行 | 红色 `#FF0000` + 加粗 |
| 普通数据行 | 黑色 `#000000` |
| 阈值线/标注（默认） | 橘色 `#ED7D31` |
