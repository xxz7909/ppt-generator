# PPT第一页（封面）详细SPEC

## 1）日期与期数规则
- 日期来源：**必须使用 Excel 文件名中的 `YYYYMMDD`**（不使用 sheet 内日期）。
- 标题日期格式：`YYYY年M月D日`（示例：`2026年2月10日`）。
- 副标题期数计算：
	- 文本格式：`统筹策划部 {YYYY}年第{day_of_year}期`
	- `day_of_year` = 该日期是当年的第几天（1月1日=1）。

## 2）标题与副标题文案
- 主标题（两行）：
	- 第一行：`{YYYY年M月D日}`
	- 第二行：`农村农业频道收视日报`
- 副标题（单行）：
	- `统筹策划部 {YYYY}年第{day_of_year}期`

## 3）字体与对齐
- 主标题字体：微软雅黑，70号，黑色 `RGB(0,0,0)`，加粗，**居中对齐**。
- 副标题字体：微软雅黑，24号，黑色 `RGB(0,0,0)`，加粗，**居中对齐**。

## 4）文本框尺寸与位置
- 主标题文本框：
	- 宽度：`26.3cm`
	- 高度：`7.3cm`
	- 当前实现位置：`left=3.8cm, top=4.8cm`
- 副标题文本框：
	- 当前实现位置：`left=8.0cm, top=12.6cm, width=17.9cm, height=1.6cm`

## 5）第一页背景与图形颜色（按当前对齐版本）
- 页面底色：浅灰（当前实现 `RGB(241,241,241)`）。
- 左上绿色圆环：`RGB(91,128,91)`。
- 左下大灰色圆：`RGB(241,243,240)`。
- 右侧灰色小圆点：`RGB(152,160,162)`。
- 右下绿色圆：`RGB(91,128,91)`。
- 图层顺序（关键）：**绿色元素必须在灰色元素之上**，即视觉上“绿色遮盖灰色”。
- 当前层级约束：左下大灰圆先绘制，左上绿色圆环后绘制；右下绿色圆位于灰色元素上层。

## 6）验收标准（第一页）
- 标题为两行，且主副标题都水平居中。
- 主标题文本框尺寸必须为 `26.3cm x 7.3cm`。
- 颜色值必须精确匹配上述 RGB。
- 日期与期数必须由文件名日期计算得出。
- 绿色圆与灰色圆的遮挡关系必须正确：**绿色覆盖灰色**。

# PPT第二页
## 1）页面用途
- 本页为“收视速报”摘要页，用 3 段正文概括当日核心变化。

## 2）标题规范
- 标题文本：`收视速报`
- 标题文本框尺寸：`宽30cm，高2cm`
- 字体：微软雅黑，32号，黑色。

## 3）正文文案结构（固定三段）
- 第1段（频道核心指标+排名）：
	- 模板：`{文件名日期月日}，频道市场份额为{当日市场份额}%，较前一个月均值{份额变化描述}；收视率{当日收视率}%，{收视率变化描述}；上星频道排名第{上星排名}位，较前一日{排名变化描述}，央视台组内排名第{台组排名}位，与前一日{台组排名变化描述}。`
- 第2段（电视剧/非电视剧）：
	- 模板：`{文件名日期月日}，对比前一个月均值，电视剧市场份额{电视剧当日份额}%，{电视剧变化描述}，非电视剧市场份额{非电视剧当日份额}%，{非电视剧变化描述}。`
- 第3段（运营分析长段）：
	- 保留业务分析文案模板，按当日内容填充。
	- 当前基准样例：
		- `{文件名日期月日}，早间节目及上午剧场收视率提升，9:24《农人秀-家乡年货开箱记》为上午剧场提供高收视入点；午间节目至黄金剧场收视率下滑，《六姊妹》后两集表现不佳，18:00《中国三农报道》将收视平台拉升至平均水平，但后续加播《振兴路上新观察》未能有效承接观众；黄金剧场时间因编排调整后移，晚间节目有效承接高收视入点，晚间及夜间节目收视提升显著。`

## 4）数据来源映射
- 第1段字段来源：
	- 当日市场份额、当日收视率、前一月均值对比：`市场份额` sheet
	- 上星频道排名：`上星频道排名` sheet（`CCTV-17农业农村`）
	- 台组排名：`台组内排名` sheet（`CCTV-17农业农村`）
- 第2段字段来源：
	- 电视剧/非电视剧当日份额及前一月均值对比：`电视剧与非电视剧` sheet

## 5）变化描述口径
- 变化幅度 `= 0`：`保持不变`
- `|变化幅度| < 5%`：`基本持平`
- `|变化幅度| >= 5%`：`提升xx%` 或 `下降xx%`

## 6）正文文本框样式
- 正文文本框尺寸：`宽30.31cm，高14.31cm`
- 字体：微软雅黑，20号，黑色。
- 第3段文字颜色：`白色，背景1，深色50%`（按模板样式）。

## 7）验收标准（第二页）
- 标题必须为“收视速报”，且尺寸/字号符合规范。
- 正文必须为 3 段结构（核心指标、电视剧对比、运营分析）。
- 数值字段全部来自指定 sheet，变化描述遵循统一口径。
